Custom Types

Custom types are a way to use Typescript to define the specific properties that are to be used on an object. This is by calling the type keyword and defining the object's properties as well as their datatypes. This is very similar to using as interface (see interface page for more information). Through this lesson we will be building a voter record type and seeing some examples of it in action. First, here is an example of the Voter type:

type Voter = {

name: string,

age: number,

registered: boolean

}


Note that the Voter object has 3 properties: name, age and registered, which have values of string, number, and boolean, respectively. To be valid it must only have those 3 specific properties and they must each have their specific data type. Here are some examples of valid and invalid Todos:


Valid

const voter1: Voter = {

name: 'Joe',

age: 25,

registered: true

}


Invalid

const voter2: Voter = {

name: 'Jessica',

age: 42,

registered: 'Yes'

}

// Notice that registered is a string instead of a boolean


Invalid

const voter3: Voter = {

name: 'Timmy',

age: 5

}

// Invalid because it is missing the registered property


Invalid

const voter4: Voter = {

name: 'Courtney',

age: 21,

registered: true,

favoriteFood: 'sushi'

}

// FavoriteFood definitely does not belong on a voter record


Once again, to correctly follow the custom type the object must have all the same properties as the interface, it must have no extra properties, and the types must match those listed in the type definition.


A type can also be used to define a function. Here is an example of a typed function:

type VoterFunc = {

(voter: Voter, voters: Voter[]): void

}


In this specific function type, the function will take in 2 parameters: voter which is a Voter type and voters which is an array of voters. It then has a void return value meaning nothing is returned. Here is an example of the implementation of our VoterFunc type:

const addVoter:VoterFunc = (voter: Voter, voters: Voter[]): void => {

voters.push(voter)

}


In this function we are taking a Voter and adding it to an array of Voters.
